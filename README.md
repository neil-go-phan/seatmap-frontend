# SEATMAP APP

- Project được viết bằng Typescript
- Sử dụng thư viện ReactJS
- Backend project:https://github.com/neil-go-phan/seatmap-backend

## HOW TO RUN
- Clone source code từ repository. 
- Mở terminal
- CD vào folder project
- Chạy các lệnh sau:
  - `npm install`
  - `npm start`
